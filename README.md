# LIC Agent Website — v1 Skeleton

Static website for an LIC Development Officer. Zero hosting cost (Vercel free tier).

## Structure
```
index.html          Home — need-based plan selector, credibility, featured plans
plans.html          All plans with filters + discontinued section
about.html          Story, milestones timeline, awards gallery
knowledge.html      FAQs + future articles
become-agent.html   Agent recruitment funnel + application form
contact.html        Enquiry form + WhatsApp
css/styles.css      All styling
js/config.js        ⭐ EDIT THIS — all personal details in one place
data/policies.js    ⭐ EDIT THIS — add/remove/discontinue plans
js/site.js          Site engine (no editing needed)
images/awards/      Put award/certificate photos here
images/team/        Team photos
```

## To go live
See DEPLOYMENT.md — full GitHub + Vercel + Formspree + domain guide.

## Quick local preview
Open index.html in any browser. Or run: `python3 -m http.server` and visit localhost:8000.
